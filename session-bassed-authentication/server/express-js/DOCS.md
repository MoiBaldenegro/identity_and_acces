# identity_and_acces
## Procedimientos:

- Instalacion de entorno
* runtime node-js + typescript
* package manager npm
* docker

# Core
npm i express express-session connect-redis ioredis prisma @prisma/client argon2 zod

# Seguridad y buenas prácticas
npm i helmet cors cookie-parser express-rate-limit hpp
npm i csurf                                   

# Logging y monitoreo
npm i winston

# Dev
npm i ts-node-dev @types/express @types/express-session @types/cookie-parser
npm i @types/cors @types/helmet

# Prisma
npm install --save-dev prisma
npx prisma init

- Configuraciones necesarios package.json, tsconfig.json o lo que corresponda.

- Estructura de carpetas

- instalar dependencias necesarias

- crear docker-compose para levantar el proyecto
